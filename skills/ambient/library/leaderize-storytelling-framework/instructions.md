# Leaderize Storytelling Framework

This skill helps you select and apply the right storytelling framework for Leaderize content, optimized for mid-level leaders experiencing functional collapse who value relief over inspiration.

## Core Principle

**Forensic empathy over persuasion.** Every framework choice must create a "How did you know?" moment for someone who hates hype and fears exposure.

## Quick Start: Framework Selection Process

### Step 1: Identify the Pain Point

Ask: **"What's the pain point this content addresses?"**

Map to ICP's hidden reality:

| **If they say...** | **Real issue** | **Journey stage** | **Framework** |
|---|---|---|---|
| "Everything escalates to me" | Escalation point problem | Awareness (5-8) | Transformation Arc |
| "AI is overwhelming" | Exposure anxiety | Search (9-12) | ABT (AI-specific) |
| "Team isn't autonomous" | Reliance pattern | Search (9-12) | Transformation Arc |
| "Can't switch off" | Emotional labor | Awareness (5-8) | Transformation Arc |
| "Need proof this works" | Trust barrier | Decision (13-16) | Case Study |
| "Leadership doesn't get it" | Authority without support | Search (9-12) | ABT (systems) |

### Step 2: Determine Content Type

**Is this written content or a live session?**

**WRITTEN CONTENT** → Continue to Step 3A
**LIVE SESSION** → Continue to Step 3B

### Step 3A: Written Content Framework Selection

Determine primary goal:

**A. CREATE "HOW DID YOU KNOW?" MOMENT**
- Reader needs to feel seen before they trust
- Use: **Transformation Arc (Nested)**
- Format: Long-form article, email sequence opener, social post
- Load: `references/transformation-arc-leaderize.md`

**B. PROVIDE CLARITY/FRAMEWORK**
- Reader needs structure to reduce overwhelm
- Use: **ABT (Clarity-Driven)**
- Format: AI explainer, systems breakdown, decision framework
- Load: `references/abt-leaderize.md`

**C. OFFER PROOF WITHOUT PRESSURE**
- Reader needs evidence but fears "too good to be true"
- Use: **Case Study Pattern**
- Format: Client story, testimonial adaptation, results showcase
- Load: `references/case-study-leaderize.md`

### Step 3B: Live Session Framework Selection

Determine session type:

**WEBINAR (20-200 people, low interaction)**
- Framework: Transformation Arc → ABT
- Load: `references/webinar-structure-leaderize.md`

**WORKSHOP (8-25 people, high interaction)**
- Framework: ABT → Application
- Load: `references/workshop-structure-leaderize.md`

**MASTERCLASS (15-50 people, medium interaction)**
- Framework: Transformation Arc → Deep Framework
- Load: `references/masterclass-structure-leaderize.md`

**COACHING (1-3 people, very high interaction)**
- Framework: Socratic ABT
- Load: `references/coaching-framework-leaderize.md`

**CONFERENCE TALK (50-500 people, very low interaction)**
- Framework: Transformation Arc → Call to Reflect
- Load: `references/transformation-arc-leaderize.md`

### Step 4: Apply ICP Alignment Check

Before proceeding, verify content meets these criteria:

- [ ] Does this create a "how did you know?" moment?
- [ ] Does it promise relief, not more work?
- [ ] Does it avoid shame or "you're broken" framing?
- [ ] Does it name the unnamed (escalation point, emotional labor)?
- [ ] Is it free from hype and inspiration theater?
- [ ] Does it position Bally/Leaderize as guide, not guru?

If any boxes are unchecked, reconsider framework choice.

## The "5 Moments" Quality Check

Before finalizing, verify content creates at least 2 of these moments:

1. **Naming escalation point** - "Everything lands with you"
2. **Framing AI as pressure** - "It's not the tool, it's the exposure"
3. **Leadership as emotional labor** - "You're the shock absorber"
4. **Calm as success metric** - "Not louder, steadier"
5. **Autonomy as relief** - "They bring options, not problems"

If it doesn't hit 2/5, reconsider framework choice.
