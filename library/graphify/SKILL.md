name: graphify
description: Turns any folder of files into a persistent, queryable knowledge graph with community detection; use for codebase/architecture questions, "graphify this", or any content query when graphify-out/ exists.
---

Read `instructions.md` in this skill's directory and follow it.

Path note: this skill also ships inside the `ambient` library plugin, so its
instructions may reference files as `${CLAUDE_PLUGIN_ROOT}/library/graphify/<file>`.
When installed standalone, resolve those to `<file>` in this directory.
