# AIMM Prompt Library

List all available `/aimm:` commands with descriptions. These prompts are extracted from AIMM Mastermind session golden nuggets and packaged as reusable tools for knowledge entrepreneurs.

## Skills

The following are full multi-step skills (not single-prompt commands). They load contextually based on the task description — invoke by describing what you need rather than typing a command.

- **eigenthinking** — Nine-step framework turning tacit expert knowledge into branded, publishable IP. UP → IP.
- **latent-cartographer** — Six-phase latent space traversal that maps the full probability landscape before converging on a non-modal answer.
- **deep-field** — Integrated three-discipline system: Eigenthinking + Latent Cartography + Wisdom of Crowds for producing non-replicable intelligence.
- **geo-page-audit** — Full GEO readiness evaluation: entity scan, structural parsability, citation competitiveness, remediation with draft copy.
- **brain-builder** — Six-step design process for encoding expertise into an AI-powered product clients use independently.
- **iteration-compressor** — Cycle mapping + compression audit that identifies exactly which workflow loops AI can collapse.
- **sourdough-excavator** — Five-round Socratic positioning interview that surfaces the hard-won, unteachable expertise that constitutes genuine differentiation.

New commands are added from AIMM session golden nuggets when they pass a 5-point inclusion rubric (actionable, discussion-native, audience-calibrated, non-obvious, reusable). To suggest a command, bring the idea to a session or message Lou.
