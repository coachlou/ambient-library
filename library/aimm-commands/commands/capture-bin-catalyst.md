# The Capture Bin Catalyst

Index and catalog a disorganized collection of notes, ideas, or fragments into a navigable, rated, thematically grouped document.

---

I have a collection of unorganized notes, ideas, and fragments in $ARGUMENTS.

Go through every item and create a single INDEX document with:

1. For each idea/note:
   - A descriptive title (create one if the original has none)
   - A 2-3 sentence summary of the core idea
   - A quality/potential rating: green (worth developing now), yellow (interesting but needs more thought), red (outdated or superseded)
   - Tags for thematic grouping

2. At the top of the index:
   - Total count of items reviewed
   - A thematic map: group related ideas together and name each cluster
   - Your top 5 recommendations for which ideas have the most potential and why

3. At the bottom:
   - Ideas that connect to each other in non-obvious ways (the interesting intersections I might have missed)

Write the index so I can scan it in 3 minutes and find any idea by theme, quality, or keyword. If any ideas are clearly variations of the same concept, consolidate them into one entry and note the duplicates.

If no folder path or collection of notes was provided above, ask me to describe or point to the collection before proceeding.
